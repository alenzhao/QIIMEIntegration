After you have grouped sequence reads into OTUs, this script will select one sequence per OTU as a representative

Here are scripts to execute before this one

  • Pick OTUs - You must have a list of OTUs with accompanying sequences in order to select one as the representative. Pick OTUs generates that list

Here are the main decisions you should make before running this script

  1. Pick an OTU list --input file (the output of Pick OTUs)
  2. Include sequence data for said OTUs. You may do this one of three ways:
    1. A reference fasta file (you used it already if you picked OTUs using uclust_ref, usearch_ref, or usearch61_ref
    2. A reference fasta file AND your input fasta file (if you allowed the picking methods above to create de novo OTUs)
    3. The input fasta file only (if you used any other OTU picking method)
  3. Specify --rep_set_picking_method.
    The default--first--is only appropriate if you picked OTUs using uclust
  4. Pick output options
    NOTE: No log file will be created if you don't specify a file path for it.

Here are the scripts to run after this one

  • Assign taxonomy - Once you have a sequence representing the whole OTU, it is feasible to determine the real world taxonomy of that OTU
