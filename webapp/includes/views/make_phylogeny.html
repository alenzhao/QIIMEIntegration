This is the final step of QIIME's phylogeny analysis. It creates a .tre phylogenetic tree file that you can import into another program for viewing/manipulation

Here are scripts to execute before this one

  • Align sequences - Input for this script is a modified fasta file. It contains headers and sequences, but some bases are replaced with -'s if they are identical to the reference alignment. The script align sequences creates such a fasta file
  • Filter alignment - Creating the tree will be much more efficient if you filter out uninformative base-pair positions

Here are the main decisions you should make before running this script

  1. Specify an --input_fp (the modified fasta file created from sequence alignment)
  2. Choose a --tree_method
    • clearcut
    • clustalw
    • fasttree_v1
    • fasttree
    • raxml_v730
    • muscle
  3. Choose a --root_method. Typically the tree_method_default will be sufficient, but in some circumstances, you may want to use midpoint instead
  4. Pick output options.
    Note: If you don't specify a --log_fp, no log file will be created

Here are the scripts to run after this one

  • None - this is the end of QIIME's phylogeny analysis

Here are some of the common pitfalls using this script

  • If you don't specify a log --file_fp, no log file will be created
