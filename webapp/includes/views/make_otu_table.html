  2. You may have assigned your OTUs to real world taxa. If so, you can include that information in the OTU table by specifying a taxonomy file (generated by the step Assign taxonomies
  3. You may also choose to exclude certain sequences from the table. To do so, upload or generate a fasta file with headers whose id's correspond to entries in the OTU table file (i.e.
    >PC.636_798 FLP3FBN01AR65G orig_bc=ACGGTGAGTGTC new_bc=ACGGTGAGTGTC bc_diffs=0
    in the fasta file corresponds to
    denovo5 PC.635_1020 PC.636_798
    in the OTU map file (i.e. only one read will be counted towards the OTU 'denovo5')).
