Once you have your OTU table, there are several useful things you can do with it. For example, you can reformat it for use with R

Here are scripts to execute before this one

  • Make OTU table - The main input for this script is the .biom formatted table that QIIME generates

Here are the main decisions you should make before running this script

  1. Pick an input file, and a name for your output
  2. Pick a command
    • summarize-table: provide OTU count and other statistics
    • convert: change file format from biom to classic (tab-delimited)
  3. Pick output options.
