This script will convert a pair of fasta/qual files (format most frequently used by QIIME) to a single fastq file (or vica versa)

Here are scripts to execute before this one

  • Demultiplex libraries - This script can also split your input into multiple output files, one per sample. Doing so only makes sense after you have demultiplexed your library
  • Demultiplex fastq
  • Join paired ends - This script only works on a single fasta/qual pair, or a single fastq file, so you may have to join your paired ends into one file before running this

Here are the main decisions you should make before running this script

  1. Are you converting from a fastq file, or to one? Pick --conversion_type
  2. Select an input sequence file, --fasta_file_path. Also pick a .qual file if converting to fastq
  3. Decide on a fastq encoding type (required for either --conversion_type), --ascii_increment
  4. Pick output options.
    • --multiple_output_files splits the input into separate output files, one per de-multplexed sample

Here are the scripts to run after this one

  • Demultiplex fastq - Demultiplex fastq is one of the only QIIME script that takes a fastq input file
