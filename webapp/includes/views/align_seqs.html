Aligning sequences to a reference is the first step in phylogeny analysis

Here are scripts to run before this one

  • Pick representative OTU sequence - This will generate a .fasta file, in which each sequence represents one OTU. Those are most likely the sequences you want to align

Here are the main decisions you should make before running this script

  1. Pick --template_fp:
    QIIME can align to any reference fasta file, but it has some built-ins, one of which you will probably want to use.
    • /macqiime/greengenes/core_set_aligned.fasta.imputed - 16S ribosomal subunit sequences for several bacterial taxa
    • /macqiime/UNITe/sh_refs_qiime_ver6_97_13.05.2014.fasta - ITS sequences for several fungal taxa
  2. Other UNITe reference sequences are present as well, with different levels of sequence similarity defining taxonomic boundaries
  3. Pick --min_length: the minimum length for an input sequence to be included in the alignment
  4. Pick an --alignment_method
    • PyNAST
    • Muscle
    • Infernal
  5. Pick alignment specific options (they will appear when you select an --alignment_method)
  6. Pick output options

Here are the scripts to run after this one

  • Filter sequence alignment - Removing base-pair positions that are not very informative will drastically improve the efficiency of phylogenetic tree creation
  • Make phylogenetic tree - Filtering is not technically necessary, so you can skip straight to tree creation
