This script takes the sequences you used to represent your OTUs and assigns them to a real-world genus, species, or other taxonomic unit.

NOTE: This is typically a very long step, and can take several minutes to complete.

Here are scripts to execute before this one

  • Pick representative OTU sequences - The sequences you will assign real-world taxa to are generated and stored in a fasta file

Here are the main decisions you should make before running this script

  1. Pick an input file (.fasta format, one sequence per OTU in your study).
  2. Pick an option for the --assignment_method parameter. The different options each have their own parameters.
    • rdp
    • blast
    • rtax
    • mothur
    • tax2tree - designed for use with Greengenes
    • uclust
  3. Pick a set of reference files
  4. Pick output options

Here are the scripts to run after this one

  • Make OTU table - The file generated by this script is an optional parameter in Make OTU table. Including it adds each OTUs taxa to the table as metadata.
