After aligning sequences for phylogenetic analysis, it is good to filter out uninformative bases. This script does just that

Here are scripts to execute before this one

  • Align sequences - This script requires an alignment, one that is in the format created by align sequences

Here are the main decisions you should make before running this script

  1. Choose your input file, a fasta file modified for sequence alignment
  2. Decide if you want to use a lanemask (a precompiled list of positions to keep or to remove). If you don't want to use one, you must use --suppress_lane_mask_filter
  3. Decide on filtering criteria, --entropy_threshold, --allowed_gap_frac, --remove_outliers/--threshold.
  4. Pick output options

Here are the scripts to run after this one

  • Make phylogeny - This script is just a preliminary step to actually creating a phylogenetic tree
